Detailed Description

The structure is responsible for converting a given query into a list of records, which satisfy it.

Note
Accepts only 'AND' and 'OR' operations

The leafs represent simple statements every other node is either 'AND' or 'OR' and its value is return is dependent on its children if one of 'AND' or 'OR' is found the recursion continues building until a simple statement is found. The tree values 'AND' nodes more than 'OR' ones. If the query consists only similar nodes, the tree will be flattened to a list.
